## Project Description:
This is a personal portfolio website built with React. The portfolio showcases my education, skills, projects, and career goals as a Computer Science student at Dalhousie University.

## Technologies Used:
- Javascript, HTML, CSS, React, Bootstrap, Vite.


### Features Implemented
1. Three Main Pages: Home, About Me, Projects.
2. Reusable Components.
3. 404 Error Handling.

### Testing Performed
- Component Testing: Verified all components render correctly.
- Routing Testing: Tested all navigation links and routes.
- 404 Testing: Confirmed unknown URLs redirect to 404 page.
- Cross Browser Testing: Tested on Chrome, Safari, and Edge.

## Installation Instructions
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run`
