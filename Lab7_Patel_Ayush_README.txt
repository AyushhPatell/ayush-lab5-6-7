## Project Description
This is my personal portfolio website built with React. For Lab 7, I added a contact form with validation, localStorage draft saving, and a messages page to display all submitted messages. The website shows my education, skills, projects, and allows visitors to contact me.

## Technologies Used
- JavaScript, React, React Router, Bootstrap, Express.js, Node.js, localStorage

## Features Implemented

### Front-End Requirements

1. Contact Form.
2. Form Validation and Sanitization.
3. Used localStorage.
4. Messages Page.

### Back-End Requirements

1. Contact Form Backend.
2. Messages Page Backend.

## API Endpoints
1. GET /api/projects - Returns all projects
2. GET /api/weather?city=Halifax - Returns weather data
3. POST /api/contact - Saves contact form data
4. GET /api/messages - Returns all submitted messages

## Testing done for contact form, localStorage, for validation, messages page, and error handling.
